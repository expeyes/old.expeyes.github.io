ExpEYES-17

The year 2017 comes with a new version of ExpEYES with some major enhancements, like increased analog sampling rate, voltage input range, programmable Sine/Triangular waveform generators, support for I2C sensors and easy to use spring loaded Input/Output connectors.
Move the cursor over the image below to know more about the features.

  • Output Connector Block.


  • Input Connector Block.


  • 1mA Constant Current Source.


  • Programmable voltage output from -5V to +5V.


  • Programmable voltage output from -3.3V to +3.3V.


  • Square wave Generator 1.


  • Square wave Generator 2. Disables WG


  • Digital Output. 0 or 5V


  • Sine/Traingular wave Generator, 5Hz to 5kHz.


  • Inverted WG.


  • Ground Terminals.


  • 0-3.3V Input, measures Capacitance.


  • 0-3.3V Input, measures frequency.


  • 0-3.3V Input,measures resistance.


  • +/-16V Input, with range selection.


  • +/-16V Input, with range selection.


  • +/-3.3V Input, gain selection by external R.


  • For external Condenser microphone


  • Gain selection resistor for A3. Rg to GND


  • Ground Terminals.


  • Select AC/DC Coupling of Input A1


  • I2C Interface for External Sensors. 5V, GND, SCL and SDA.


  • +/-6V/10mA dual supply, for Op-Amps


  • Status LED, indicating the Communication.


  • USB Connector on the left side.


The software has been modified to utilize the extra features of the new hardware. The main GUI window consists of a four channel oscilloscope, and widgets to control/monitor the other features like waveform generators, voltage sources, frequency counter etc. Programs for various experiments can be started from this main window. Move the cursor over the image below to know more about the GUI

  • Measures the capacitance connected between IN1 and GND.


  • Measures the frequency of a TTL level signal connected to IN2.


This version of ExpEYES is based on the work released under GPL by Cspark Research. We have modified the TkInter based GUI programs of ExpEYES-Junior to run on this. Python-Qt based GUI programs are made available by CSpark.
