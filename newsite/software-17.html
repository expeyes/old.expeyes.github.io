Powered by Python

The ExpEYES communication library and graphical applications are written in Python: a cross-platform, high-level language. Packages have been made for Linux, Windows, and OSX, but it should typically run on any platform supporting Python as long as dependencies are met

  • Bootable ISO for all systems
  • Debian / Ubuntu
  • Windows 7/10
  • Other GNU/Linux distros

The expEYES Live ISO images for DVDs and USB pendrives

The fastest way to get expEYES running on your PC is to boot your computer from a DVD or USB pendrive, prepared using the ISO image (Download Link).   The procedure to do this on MSWindows is explained here. Once you boot from the pen-drive, the Lubuntu desktop will appear.
Start the expEYES program from the Applications->Science menu.

This ISO image is prepared from Lubuntu 14.04 and contains other educational resources also.

Software for Debian/Ubuntu

Added on Feb-2017 : For running ExpEYES-17, download eyes17-1.0.0.deb and install using the command

$ sudo gdebi eyes17-1.0.0.deb

On MS Windows

Since the programs are written in Python, the same source code works on GNU/Linux and Windows. You need to install Python Interpreter and the required libraries.

You will need to install the following Python libraries :

  • Python Interpreter version 2.7xx (It is also Python3 compatible
  • Python Serial module
  • Python TkInter module
  • Python Imaging Library module
  • Scipy and Numpy modules
  • MCP2200 USB device driver if not automatically detected

Download expeyes-5.0.0.zip, extract it to a suitable location, and then run the program scope17.py

Other GNU/Linux distributions

Download expeyes-5.0.0.zip and execute the following:

$ unzip expeyes.zip
$ cd expeyes-5.0.0
$ sudo python scope17.py

To use the device as a normal user, download the file to set the USB permissions and execute it:

# sh postinst.sh
