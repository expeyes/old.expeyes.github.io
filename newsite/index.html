Device Features: ExpEYES-17

  • Fully functional Python API
  • 4-Channel Oscilloscope
  • 8MHz Timing Measurement
  • +/-16V to +/-10mV voltage ranges
  • Capacitance & Resistance Measurement
  • Over 50 experiments

Fully controllable via the Python Library and GUI

Over 50 different experiments with dedicated graphical user interfaces, Schematic diagrams, and help files are ready to use. The individual function calls can also be accessed from Python scripts, and several code examples are included. The code is compatible with Linux, Windows & OSX
